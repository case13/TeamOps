# TeamOps

Delphi Horse API for service and team management using MVC.

## Overview

TeamOps is a public portfolio project built with Delphi and Horse, designed to demonstrate backend development skills for international opportunities.

The project simulates a service and team management system, where it is possible to manage teams, employees, services, tasks, and operational workflows through a REST API.

Its main purpose is to showcase clean project organization, MVC structure, REST principles, and maintainable Delphi backend code.

## Tech Stack

- Delphi
- Horse
- MVC architecture
- REST API
- JSON

## Main Features

- Team management
- Employee management
- Service registration
- Task and assignment control
- Operational workflow organization
- REST endpoints for CRUD operations

## Project Goals

This project was created as part of my backend portfolio to demonstrate:

- Delphi backend development
- API design with Horse
- MVC-based project structure
- Clean and organized code
- Readable and maintainable architecture

## Folder Structure

The project is organized using MVC principles, separating responsibilities into clear layers such as:

- Controllers
- Models
- Services
- Repositories
- Routes
- DTOs
- Utils
